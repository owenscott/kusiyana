Kusiyana - Chichewa Spelling Permutations

Chichewa spelling has a lot of variation in daily usage. This variation makes data collection and data lookup difficult when a Chichewa word, such as a village name, is being used as a key. This module generates simple permutations for Chichewa words, based on common spelling variations. The permutations can then be used as keys in a lookup function. Essentially it enables sort of a Chichewa-specific fuzzy search. The form, below, gives a basic demo.
